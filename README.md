# Система управления рекламными экранами AdvertControl

Сайт проекта: **[advertcontrol.ru](https://advertcontrol.ru)**

---

## Описание проекта

Система предназначена для управления содержимым рекламных и информационных экранов в общественных местах (кинотеатры, рестораны и т.д.). 
Через web-интерфейс формируются конфигурации (конфиги) для экранов. Конфиг передаётся на конечное устройство (приложение). Приложение — кроссплатформенное. Поддерживаются разные ОС экранов: Android TV, Linux, Windows.
Цель: обеспечить централизованное управление контентом, надёжную доставку конфигов и простое масштабирование сети экранов.

## Основные возможности

* Управление экранами и их конфигурациями через web-интерфейс.
* Формирование и версия конфигов.
* Привязка физического устройства к конфигу (pairing / QR).
* Поддержка типов контента: Image, Video, Inline JSON (таблицы).
* Кэширование медиа на устройстве с проверкой контрольных сумм.
* Высокая доступность и fallback: REST → gRPC.
* Аутентификация и RBAC через Keycloak.
* Хранение файлов в MinIO.
* Локальная сборка и отладка в Docker / docker-compose.
* Кроссплатформенность

---

## Архитектура (вкратце)

* Backend API (C#, .NET) — gRPC + REST. OpenAPI/Swagger.
* База данных — PostgreSQL (code-first миграции).
* Хранилище медиа — MinIO (S3-совместимое).
* Кэш / очереди — Redis (кеш конфигов, быстрая выдача).
* Аутентификация — Keycloak (realm/clients/roles настроены автоматически).
* Клиент — кроссплатформенное приложение (C#, Avalonia) + Android клиент.
* CI/CD и развёртывание — контейнеры Docker, конфигурации docker-compose, готовые пайплайны для CI.

## Технологии

Бэкэнд:

* Язык: C#
* Платформа: .NET (API, сервисы)
* Клиент: Avalonia (кроссплатформенно), Android (native часть)
* БД: PostgreSQL
* Хранилище файлов: MinIO
* Протоколы: gRPC, REST (OpenAPI/Swagger)
* Кэш: Redis
* Аутентификация: Keycloak
* Контейнеризация: Docker, docker-compose
* QR-генерация: ZXing / SkiaSharp (на клиенте)

Фронтэнд: 

* Язык: TypeScript
* Фреймворки: React, Tailwind CSS
* Сборщик: Vite
* Менеджер состояний: Redux
* Rest-запросы: axios
* Библиотека компонентов: Radix UI
* Drag & Drop: dnd-kit
* Рендеринг графики: WebGL, Three.js
* PWA: vite-plugin-pwa
